# Salam Hammoud Portfolio

موقع شخصي احترافي للإعلامية سلام حمود، مبني باستخدام HTML, Tailwind CSS, و Vanilla JS.

## المميزات
- **تصميم عصري:** وضع ليلي (Dark Mode) مع لمسات ذهبية فاخرة.
- **مبني على البيانات (Data-Driven):** جميع النصوص والصور والروابط موجودة في ملف `data/data.json`.
- **لوحة تحكم (Admin Panel):** لتعديل المحتوى بسهولة دون لمس الكود.
- **سريع ومتجاوب:** يعمل على جميع الأجهزة.

## طريقة التشغيل (Locally)
بسبب سياسات الأمان في المتصفحات (CORS)، لا يمكن قراءة ملف `data.json` عند فتح ملف `index.html` مباشرة بالنقر المزدوج. يجب تشغيله عبر خادم محلي.

### باستخدام VS Code:
1. قم بتثبيت إضافة **Live Server**.
2. اضغط بزر الفأرة الأيمن على `index.html` واختر **Open with Live Server**.

## لوحة التحكم
1. اذهب إلى `admin.html`.
2. كلمة المرور الافتراضية: `admin123`.
3. قم بتعديل البيانات كما تريد.
4. اضغط "حفظ وتحميل JSON".
5. استبدل ملف `data/data.json` القديم بالملف الجديد الذي تم تحميله.
6. ارفع التغييرات إلى GitHub.

## النشر على GitHub Pages
1. ارفع المجلد `salam-portfolio` إلى مستودع GitHub.
2. اذهب إلى Settings > Pages.
3. اختر الفرع `main` والمجلد `salam-portfolio` (أو `root` إذا رفعت المحتويات مباشرة).
4. سيتم نشر الموقع وسيعمل ملف `data.json` بشكل طبيعي.
